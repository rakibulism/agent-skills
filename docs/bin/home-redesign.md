# Designer Brief: Restyle agent-skills → Mobbin Style

## What to do

Take the current **agent-skills** website and restyle it to match the visual design language of **Mobbin**. 

**All content stays exactly the same.** Do not change any text, links, labels, tags, or data. Only the visual style and layout structure changes.

---

## Key changes to make

### 1. Search bar
- Move the search bar to the **top center of the page**, just like Mobbin
- Make it **full-width**, max ~720px, horizontally centered
- Style it as a **pill shape** with a magnifier icon inside on the left
- Light gray background, no heavy border
- The search bar becomes the hero — remove the large headline above it

### 2. Navigation
- Keep: logo left, nav links center, GitHub/X buttons right
- Style the nav with a **thin bottom border only** (`0.5px`)
- GitHub and X buttons should be **pill-shaped** with subtle background + small icon + label

### 3. Browse categories section
- Below the search bar, add a **4-column grid** with column headers:
  `Categories` / `Screens` / `UI Elements` / `Flows`
- Each column header in **tiny all-caps muted text**
- Links underneath in **large, bold typography** — editorial style like Mobbin

### 4. Tabs bar
- Tabs: Featured / Most popular / Top rated / New
- Active tab has a **bottom border underline indicator**
- Filter button sits **flush right**
- Separated from content by a thin border

### 5. Cards
- Switch to **compact cards** with:
  - Category badge (pill, gray) top-left
  - Bold title
  - `#hash` subtitle line
  - Tag chips row (`#node`, `#api`, etc.)
  - Preview + Download actions at the bottom, separated by a thin top border
- **4-column grid layout**

### 6. Typography & Color
- Heavy grotesque font for headings (e.g. DM Sans Bold or similar)
- Clean regular weight for body/tags
- **Near-monochrome palette only** — black, white, and grays
- No decorative colors, no gradients
